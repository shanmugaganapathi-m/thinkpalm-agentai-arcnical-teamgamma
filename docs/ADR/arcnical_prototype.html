Arcnical
Architecture Needs Intelligence
Arcnical is an AI-powered GitHub repository analyzer that performs multi-layered architectural analysis — from static qualification through LLM-backed deep review. Built as a 7-day prototype sprint, it delivers actionable insights into code health, dependency graphs, security surface, and maintainability.
⚙
Tech Stack
Python 3.11+ · uv package manager
tree-sitter (Python + TypeScript parsers)
networkx · PyVis (dependency graphs)
Pydantic v2 (600+ lines schema)
Streamlit · Rich · Typer (UI/CLI)
Jinja2 (report templates)
Anthropic SDK · OpenAI · Gemini (LLMs)
🧠
Analysis Layers
L1 — Qualification (static triage)
L2 — Heuristics (circular deps, complexity)
L3 — Security (secrets, CVE surface)
L4 — LLM Review (deep architectural insight)
